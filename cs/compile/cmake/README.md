# 食用指南
在工作中如果需要新建cmake工程，可以根据使用需求，在使用场景中找到对应的案例进行模仿。

对于常用命令如果不熟悉，可以查看命令要点说明。这里不会对命令进行面面俱到的说明，只会列出常见的使用注意点，没有涉及到的情况可以查看官方说明文档。

如果需要系统性地练习cmake，可以参照github上他人的或官方的学习引导案例。
