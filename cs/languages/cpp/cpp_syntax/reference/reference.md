# 引用的用法
## 对象引用
这里的引用其实是左值引用. 到c++11之后又多了一个右值引用. 
引用其实就是给变量起别名
```cpp
int a = 5;
int &ra = a;
```

## 数组引用
有两种方式
1. 直接定义并初始化
1. 使用typedef定义数组类型, 然后用定义的类型初始化引用

```cpp
int arr[2] = {1, 2};
//1
int (&rarr)[2] = arr;
//2
typedef int(ARR_REF)[2];
ARR_REF &rarr2 = arr;
```

## 多态用法
可以和指针一样, 实现多态调用

使用时的要求
1. 必须初始化引用的变量, 并且该引用之后不允许修改为引用其他变量
1. 必须引用合法的内存空间
1. 不要返回局部变量的引用. 如果要返回引用, 这个返回值需要是全局或静态变量

# 引用的本质
本质就是指针常量, 必须初始化指向, 之后不可修改. 只是在取值的时候不需要通过指针引用的方式, 而是像变量一样. 
```cpp
int a = 5;
int *const p3 = &a; //引用方式*p3, 这里const修饰的是p3
int &ra = a; //引用方式ra
```

# 应用场景
1. 作为函数的传入传出参数

    指针也可以当作引用作为函数的传入传出参数, 类似于函数参数使用二级指针. 可以在函数中改变指针的指向, 例如分配空间等.
    ```cpp
    void func(int *&p) {
        p = new int(50);
    }
    ```

1. 常量引用

    类似const int *const p的写法. 对于结构体的常量引用, 可以声明数据为只读, 并且不额外开辟空间(不会重新构造).
